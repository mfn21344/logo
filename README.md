## 一些实用小技巧

如果你打算用Logo.com，这几个技巧可以帮你做出更好的设计：

**1. 多试几轮关键词**
输入品牌信息时，试着用不同的描述词。比如你做咖啡店，除了输入"coffee"，还可以试试"espresso"、"artisan coffee"、"cozy cafe"。不同词汇会触发不同的设计风格。

**2. 从简单开始**
AI生成的设计里，往往越简单的越经典。那些看起来"很炫"的复杂设计，过两年可能就过时了。简洁的Logo更容易在小尺寸（比如手机图标）上保持辨识度。

**3. 注意颜色心理**
不要只凭喜好选颜色。蓝色传达信任和专业（银行、科技公司常用），红色代表活力和激情（快餐、运动品牌），绿色象征健康和环保。Logo.com的配色建议通常已经考虑了行业特点，但你也可以根据自己的品牌调性调整。

**4. 下载所有格式**
免费版就能下载PNG、JPG、SVG，记得都存一份。SVG是矢量格式，以后不管是印在名片上还是做成巨型广告牌，放大缩小都不会失真。

**5. 保留多个版本**
Logo.com允许你保存多个设计方案。如果拿不定主意选哪个，可以先保存几个候选，然后发给朋友、客户看看反馈，再做最终决定。
